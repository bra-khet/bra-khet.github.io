What Happened

The Leak

A single CMS visibility toggle left public exposed ~3,000 internal Anthropic assets on March 26, 2026. Investigative reporter Bea Nolan of Fortune found the unsecured directory and authenticated the contents with two independent researchers.

"Human error in the CMS configuration." — Anthropic spokesperson

The Model

Claude Mythos (codename: Capybara) is a fourth, super-tier AI above Opus — designed for autonomous agentic ecosystems, competition-level math, and frontier software engineering. Anthropic engineers called it "by far the most powerful AI model we've ever developed."

Recursive development loop: built using Claude Opus 4.6, then folded back in to build its successors.

The Threat

Internal assessments warn Mythos is "currently far ahead of any other AI model in cyber capabilities." Anthropic's Frontier Red Team documented 500+ zero-day vulnerabilities found autonomously by an earlier Opus model. Mythos represents a qualitative leap beyond that baseline.

"Presages an upcoming wave that can exploit vulnerabilities far outpacing defenders."
