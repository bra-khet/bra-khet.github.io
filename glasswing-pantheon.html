Their Justification

◆ The Glasswing Doctrine

  • Mythos can autonomously discover and weaponize zero-day vulnerabilities at machine speed — too fast for defenders using current tools.
  • General API availability would instantly democratize advanced cyber-warfare capabilities to hostile nation-states, ransomware syndicates, and script kiddies with an API key.
  • A defensive-first consortium compresses vulnerability triage from months of human labor to minutes of machine computation — giving defenders the advantage, briefly.
  • RSP v3.1 abandoned the pause commitment: competitive necessity now outweighs voluntary deceleration. The arms race is real and they're not stopping.
  • Once open-source equivalents inevitably proliferate to adversarial actors, the window for this advantage closes permanently.
